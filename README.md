## User Guide

### how to contribute

```shell
# download from gitlab
$ git clone ssh://git@gitlab.mykg.tech:2224/mykg/genius/js-norm.git

# add github remote
$ git remote add github git@github.com:mykg-ai/js-norm.git
```

### how to serve in local

[1] use docsify

```shell script
$ docsify serve html
```

[2] use serve

```shell script
$ serve -s html
```

### how to deploy

```shell
$ sh release.sh
```
